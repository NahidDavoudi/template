# Smoke Test — دروازه لانچ MVP

## معیار Go/No-Go

| معیار | وضعیت |
|-------|--------|
| بدون blocker امنیتی باز | ☐ |
| بدون ریدایرکت شکسته | ☐ |
| بدون داده جعلی در UI پرداخت | ☐ |
| بدون خطای 5xx تکراری در لاگ | ☐ |

## 1. ورود / ثبت‌نام

- [ ] باز کردن `login.html`
- [ ] ثبت‌نام کاربر جدید یا ورود با OTP
- [ ] ریدایرکت به `index.html#/` بعد از login
- [ ] تلاش ۶ بار login اشتباه → پیام rate limit (429)

## 2. فروشگاه

- [ ] `#/` — صفحه اصلی بدون خطای console
- [ ] `#/shop` — لیست محصولات
- [ ] `#/product?id=X` — جزئیات محصول
- [ ] افزودن به سبد از کارت محصول

## 3. سبد و checkout

- [ ] `#/cart` — نمایش آیتم‌ها
- [ ] `#/checkout` — بدون login ریدایرکت به login
- [ ] بعد از login — تکمیل checkout و ثبت سفارش

## 4. پرداخت کارت‌به‌کارت

- [ ] `#/payment` — شماره کارت واقعی از تنظیمات (نه fallback)
- [ ] آپلود رسید (jpg/png/webp/pdf)
- [ ] پیام موفقیت و redirect

## 5. سفارشات کاربر

- [ ] `#/orders` — لیست سفارشات کاربر

## 6. پنل ادمین

- [ ] `admin.html` — login ادمین
- [ ] مشاهده سفارش جدید
- [ ] تایید/رد رسید
- [ ] ویرایش محصول (General tab)

## 7. قابلیت‌های غیرفعال MVP

- [ ] بخش خبرنامه در صفحه اصلی نمایش داده **نمی‌شود**
- [ ] فرم تماس فقط اطلاعات تماس نشان می‌دهد (بدون submit جعلی)

## تصمیم نهایی

- **GO** — همه موارد حیاتی ✅
- **NO-GO** — هر blocker امنیتی یا مسیر خرید شکسته ❌
