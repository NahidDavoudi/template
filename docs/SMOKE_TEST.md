# Smoke Test — دروازه لانچ MVP (React SPA)

## معیار Go/No-Go

| معیار | وضعیت |
|-------|--------|
| بدون blocker امنیتی باز | ☐ |
| بدون ریدایرکت شکسته | ☐ |
| بدون داده جعلی در UI پرداخت | ☐ |
| بدون خطای 5xx تکراری در لاگ | ☐ |

## 0. Build

- [ ] `npm run build` بدون خطا خروجی `dist/` تولید کند
- [ ] `npm run preview` روی `http://localhost:4173` باز شود

## 1. ورود / ثبت‌نام

- [ ] باز کردن `/login`
- [ ] ثبت‌نام کاربر جدید یا ورود با OTP (اگر smsOtpEnabled)
- [ ] ورود نمایشی ادمین با `admin / admin1234` و ریدایرکت به `/admin`
- [ ] ریدایرکت به `/` (یا `redirect=`) بعد از login موفق
- [ ] تلاش ۶ بار login اشتباه → پیام rate limit (429)

## 2. فروشگاه

- [ ] `/` — صفحه اصلی بدون خطای console، کاروسل و پوسترها
- [ ] `/shop` — لیست محصولات با فیلتر دسته و مرتب‌سازی
- [ ] `/product/:id` — جزئیات محصول، انتخاب واریانت، افزودن به سبد
- [ ] `/categories` — نمایش دسته‌بندی‌ها

## 3. سبد و checkout

- [ ] `/cart` — نمایش آیتم‌ها (مهمان و کاربر)
- [ ] `/checkout` — بدون login ریدایرکت به `/login?redirect=/checkout`
- [ ] بعد از login — انتخاب/افزودن آدرس و ثبت سفارش

## 4. پرداخت کارت‌به‌کارت

- [ ] `/payment/:id` — شماره کارت واقعی از تنظیمات (نه fallback)
- [ ] آپلود رسید (jpg/png/webp/pdf)
- [ ] پیام موفقیت و redirect

## 5. سفارشات و پروفایل کاربر

- [ ] `/orders` — لیست سفارشات کاربر + مشاهده جزئیات
- [ ] `/profile` — مدیریت آدرس‌ها (افزودن/ویرایش/حذف/پیش‌فرض)

## 6. پنل ادمین

- [ ] `/admin` — بدون ادمین ریدایرکت به `/login`
- [ ] داشبورد: آمار و نمودارها بارگذاری شود
- [ ] محصولات: لیست + ویرایشگر محصول (تب‌های عمومی/واریانت/تصاویر)
- [ ] دسته‌بندی‌ها: افزودن/ویرایش/حذف + آپلود پوستر
- [ ] سفارش‌ها: تغییر وضعیت، تایید/رد رسید
- [ ] کاربران: تغییر نقش، حذف
- [ ] تخفیف‌ها: ایجاد/غیرفعال/حذف
- [ ] پوسترها: آپلود، مرتب‌سازی، فعال/غیرفعال
- [ ] تنظیمات: تب‌های هویت/پرداخت/تماس/ارسال/پیامک/سئو
- [ ] محتوای صفحات: ویرایش صفحات استاتیک

## 7. قابلیت‌های غیرفعال MVP

- [ ] بخش خبرنامه در صفحه اصلی نمایش داده **نمی‌شود**
- [ ] فرم تماس فقط اطلاعات تماس نشان می‌دهد (بدون submit جعلی)

## تصمیم نهایی

- **GO** — همه موارد حیاتی ✅
- **NO-GO** — هر blocker امنیتی یا مسیر خرید شکسته ❌
